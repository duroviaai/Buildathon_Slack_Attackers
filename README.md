# Buildathon - Slack Attackers

An AI-powered personalized learning platform built for the Buildathon hackathon.

## Project

**MindForge** — an adaptive learning web app that helps students learn smarter using AI.

## Features

- 🤖 AI Chatbot (SambaNova)
- 🃏 Flashcard generation
- 📝 Quiz generation
- 📓 Notes management with OCR support
- 🔥 Learning streaks
- 👤 Learner profiling
- 🧑‍🏫 Teacher dashboard
- 🔐 Auth with Google OAuth + JWT + OTP

## Tech Stack

- **Backend:** Python, Flask, SQLAlchemy, Flask-JWT-Extended
- **Frontend:** HTML, Tailwind CSS, Vanilla JS
- **AI:** SambaNova API (OpenAI-compatible)
- **Database:** SQLite
